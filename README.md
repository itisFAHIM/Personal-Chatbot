# FAHIM_Code_Korbi

A local AI coding assistant built with **Django** and **Ollama (Gemma 3:4b)**. This agent mimics a senior developer persona, capable of debugging code and maintaining conversation history locally.

## Features
- **Local AI:** Runs completely offline using Ollama in user's PC.
- **Context Memory:** Remembers previous chat turns.
- **VS Code Style UI:** Dark mode with syntax highlighting for code blocks.
